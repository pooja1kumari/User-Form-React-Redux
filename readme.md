# React Redux User Add and Edit

### Database 
- localStorage

### Programing language
- React 
- TypeScript
For UI, Material-UI is used.

### How to run project
1. Go to project's root directory.
2. `bash run_app.sh`
3. wait for database start complete
5. Application will be served at http://localhost:3000
